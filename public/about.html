croc

Quick access to your locally installed rdocs.

Examples with rspec

test Gems, classes and methods that contain test
test_finished. Classes and methods named exactly test_finished
rsp test Classes and methods that contain test in gems that contain rsp
rsp test_finished. Classes and methods named exactly test_finished in gems that contain rsp

Install your own rdocs

Copy your rdocs directory to ~/.croc/rdoc and make sure it's named something sensible. Rerun croc.
